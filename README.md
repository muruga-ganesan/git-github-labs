# Git & GitHub Labs

## Overview

This repository contains my hands-on Git and GitHub learning journey. It includes practical exercises, real-world troubleshooting scenarios, SSH authentication, version control workflows, branching strategies, collaboration techniques, and best practices used by DevOps, Site Reliability Engineering (SRE), and Production Support teams.

The objective of this repository is to build practical Git knowledge while maintaining my engineering portfolio.
